# Описание настроек стилей

## Секция app
- `window_size`: Размер окна приложения (ширина x высота)
- `theme`: Тема приложения - "Light", "Dark" или "System"
- `color_theme`: Цветовая схема - "blue", "green", "dark-blue"

## Секция colors
- `primary`: Основной цвет кнопок (синий)
- `primary_hover`: Цвет кнопок при наведении
- `danger`: Цвет опасных действий (красный)
- `danger_hover`: Цвет опасных кнопок при наведении
- `accent`: Акцентный цвет (оранжевый)
- `accent_hover`: Акцентный цвет при наведении
- `background`: Фон элементов
- `tab_hover_dark`: Цвет вкладок при наведении (темная тема)
- `tab_hover_light`: Цвет вкладок при наведении (светлая тема)

## Секция fonts
- `title`: Настройки шрифта заголовка (размер и насыщенность)
- `tabs`: Настройки шрифта вкладок
- `buttons`: Настройки шрифта кнопок
- `input`: Настройки шрифта полей ввода
- `content`: Настройки шрифта основного контента

## Секция sizes
- `tab_button_height`: Высота кнопок вкладок
- `standard_button_height`: Стандартная высота кнопок
- `large_button_height`: Высота больших кнопок
- `add_button_width`: Ширина кнопки "Добавить"
- `tabs_frame_width`: Ширина панели с вкладками
- `report_text_height`: Высота поля отчета
- `plans_text_height`: Высота поля планов

## Секция padding
- `main_frame`: Отступы основного контейнера
- `header_pady`: Вертикальные отступы заголовка
- `standard_pady`: Стандартные вертикальные отступы
- `small_pady`: Маленькие вертикальные отступы
- `button_padx`: Горизонтальные отступы между кнопками
- `tabs_padx`: Горизонтальные отступы вкладок

## Секция icons
- Эмодзи-иконки для визуального оформления всех элементов

## Секция texts
- Все текстовые надписи в приложении на русском языке
